# 🕒 2026 Time Visualizer

A minimal and elegant **React Native app** that visualizes the passage of time throughout the year 2026 using clean dot-based grids.

It helps you *see time* — not just track it.

---

## ✨ Features

- 📅 **Real-time Year Progress**
  - Displays how much of the year has passed

- 🔢 **Multiple Time Views**
  - **Now / Today** – Remaining days
  - **Month** – Days per month
  - **Week** – 52-week grid
  - **Year** – Full year overview
  - **Stats** – Live countdown (seconds → months)

- ⏱ **Live Updates**
  - Updates every second automatically

- 🎯 **Minimal UI**
  - Clean dark theme
  - Sidebar navigation
  - Dot-based visualization system
